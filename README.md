# 🌸 Shishu-Sneh  
### Maternal & Child Healthcare Android Application

Shishu-Sneh is an Android-based healthcare application developed during the **Android App Development using Generative AI Internship at MindMatrix**.  
The application is designed to assist new mothers by providing essential healthcare support for babies during their first year of life. It includes features such as baby growth tracking, vaccination reminders, feeding guidance, and milestone monitoring in a simple and user-friendly interface.
The project mainly focuses on improving maternal awareness and child healthcare accessibility, especially for users in rural and semi-urban regions.

# 📱 Application Features

## 👶 Baby Profile Setup
- Store baby details such as:
  - Name
  - Date of Birth

## 📊 Growth Tracking
- Track baby:
  - Weight
  - Height
- Visual growth monitoring using charts
- Easy progress analysis

## 💉 Vaccination Reminder System
- Automatically generates vaccination schedules
- Timely reminder notifications
- Prevents missed vaccinations

## 🥗 Feeding Guide
- Personalized feeding and nutrition suggestions
- AI-powered recommendations

## ✅ Milestone Monitoring
- Track developmental milestones
- Simple yes/no checklist system
- Monitor baby progress regularly

## 📦 Offline Functionality
- Local data storage using Room Database
- Works without internet connection

## 🎨 User-Friendly UI
- Soft pastel themed design
- Clean and minimal interface
- Designed for non-technical users

# 🏗️ Application Architecture

The application follows a modular architecture for better scalability and maintainability.

## 🔹 UI Layer
Handles screens and user interactions with a clean and responsive design.

## 🔹 Navigation Layer
Manages smooth navigation between:
- Onboarding
- Dashboard
- Growth Tracker
- Vaccination Module
- Feeding Guide
- Milestone Tracker

## 🔹 Data Layer
Uses Room Database for secure local data storage.

## 🔹 Background Processing Layer
Uses WorkManager for:
- Notifications
- Vaccination reminders
- Background task scheduling

## 🔹 AI Integration Layer
Uses Generative AI to provide:
- Personalized feeding tips
- Intelligent healthcare guidance
